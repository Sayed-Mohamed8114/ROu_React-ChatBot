#  Rou – AI Chat Application

Rou is a modern AI-powered chat application inspired by Google's Gemini. Built with **React** and powered by the **Google Gemini API**, it provides a clean, fast, and responsive interface that allows users to interact with an AI assistant in real time.

##  Features

* Real-time conversations powered by Google Gemini AI
* Streaming responses for a more natural chat experience
* Clean and intuitive user interface
* Fully responsive design for desktop and mobile devices
* Markdown rendering with syntax highlighting for code snippets
* Loading states and error handling for a smooth user experience
* Component-based architecture for maintainability and scalability

##  Technologies Used

* React
* JavaScript (ES6+)
* Google Gemini API
* Tailwind CSS
* React Markdown
* React Syntax Highlighter
* Vite

##  Installation

Clone the repository:

```bash
git clone https://github.com/your-username/rou.git
cd rou
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the project root:

```env
VITE_API_KEY=your_gemini_api_key
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Rou was built to explore modern AI application development using React and the Gemini API, focusing on creating a simple, responsive, and enjoyable conversational experience while practicing real-time streaming responses and clean frontend architecture.
